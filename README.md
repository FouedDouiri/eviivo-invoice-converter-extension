# Eviivo Invoice Converter Extension

A browser extension that extracts invoice data from Eviivo and recreates the invoice with prices converted into Tunisian Dinar (TND).

## Problem

Eviivo invoices are generated in foreign currencies (EUR / GBP), which makes accounting difficult for Tunisian businesses that must operate in TND.

## Solution

This extension:

• Scrapes invoice data directly from the Eviivo interface  
• Extracts line items and totals  
• Converts prices to Tunisian Dinar  
• Recreates a clean invoice in TND for accounting purposes  

## Features

- Automatic invoice data extraction
- Currency conversion
- Invoice recreation in TND
- Simple one-click workflow

## Tech Stack

- JavaScript
- Browser Extension API
- DOM scraping
- Currency conversion logic
